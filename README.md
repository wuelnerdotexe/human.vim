# Vim for Humans.

Default vim options collection for humans. No further description is needed.

## Installation

Install via your preferred package manager. Example using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'wuelnerdotexe/human.vim'
```

IMPORTANT: Human should be the first plugin on the list.
